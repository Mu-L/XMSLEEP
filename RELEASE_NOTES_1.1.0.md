# XMSLEEP v1.1.0 发行说明

## ✨ 新功能

- **空状态动画**：为空白页面添加了精美的动画提示
- **批量选择**：支持批量设置默认播放区域，操作更便捷

## 🎨 UI 优化

- **默认播放区域**：添加使用说明，限制最多3个卡片
- **统一间距**：优化各区域间距，界面更整洁
- **按钮优化**：播放按钮使用强调主题色，更醒目
- **编辑模式**：优化编辑按钮样式和批量选择状态显示

## 🔧 改进

- 更新关于对话框使用说明
- 修复废弃图标警告

## 📦 版本信息

- **版本号**: 1.1.0
- **Version Code**: 10
